# Croupier

Collection of sampling strategies for the Elementary Effects method.

## Install

### uv

    $ uv add git+https://github.com/gabrielbdsantos/croupier.git

### Pip

    $ pip install git+https://github.com/gabrielbdsantos/croupier.git@master
