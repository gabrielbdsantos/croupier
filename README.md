# Croupier

A minimal sampling framework for the Elementary Effects method.

## Install

### Poetry

    $ poetry add git+https://github.com/gabrielbdsantos/croupier.git

### Pip

    $ pip install git+https://github.com/gabrielbdsantos/croupier.git@master
